hg-engine is a comprehensive engine upgrade for Pokémon HeartGold.
It aims to implement features that have come since the release of the Nintendo DS in future generations, from Pokémon Black and White all the way through Scarlet and Violet.

This involves brand new Pokémon, abilities, moves, and miscellaneous new mechanics.

Code Injection Framework

hg-engine is powered by a code injection framework that parses a text file to insert hooks into the ARM binaries and take control of program flow.
